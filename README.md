## [RUN](./.github/workflows/cmake.yml)

1. clone `git clone https://github.com/DrAugus/cpp.git`
2. enter the dir `cd cpp`
3. make compile dir and enter `mkdir cmake-linux-debug && cd cmake-linux-debug`
4. cmake `cmake ..`
5. make `make`
6. run `./augus_cpp`
